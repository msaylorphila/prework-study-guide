HTML

  • The head element contains information about the webpage.
  • The body element contains the visible content shown to the user.
  • The footer element appears at the bottom of the page and ususally contains the author, copyright, contact, sitemap, and navigation.

CSS

  • A margin indicates how much space we want around the outside of an element.
  • A padding indicates how much space we want around the content inside the element.

Git

  • git status: checks what branch we are currently on
  • git checkout -b branch-name:creates a new branch and switches to it

JavaScript

  • JavaScript is case sensitive
  • Variables are containers that store values
  • Comments are snippets of text that can be added along with code
  • Conditionals are code structures used to test if an expression returns true or not
  • Functions are a way of packaging functionality that you wish to reuse
  • Control flow is the order in which a computer executes code in a script
